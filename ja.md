# Phrabbit ユーザーガイド

🌐 [한국어](ko.md) · [English](en.md) · **日本語** · [简体中文](zh-Hans.md) · [繁體中文](zh-Hant.md)

A/B 区間リピート再生で外国語を練習する iOS アプリです。音声ファイルを読み込み、波形の上で好きな区間を自由にリピートし、話されている内容を自動で文字に起こして、聞きながら一緒に読むことができます。

## 1. はじめに

### 1-1. 初回起動 — オンボーディング
アプリを初めて開くと、3 枚の紹介スライドが表示されます。

1. **波形で区間を正確に選択** — 音声をグラフで見ながら、聞きたい部分をピンポイントで指定できます。
2. **A と B の間を無限リピート** — 開始点（A）と終了点（B）を設定すると、その区間だけが自動で繰り返されます。
3. **リアルタイム音声テキスト変換** — 聞いている音声が自動で文字に起こされ、一緒に読めます。

右上の **「Skip」** ボタンでスキップするか、最後のスライドの **「Get Started」** ボタンで開始します。

![オンボーディングスライド](images/01-onboarding.PNG)
*▲ `images/01-onboarding.PNG` — オンボーディングの最初のスライド（波形アニメーション付き）*

### 1-2. 無料トライアル
初回起動時に **3 日間の無料トライアル** が自動で始まります。3 日間はすべての機能を自由に試せます。トライアル中はプレーヤー上部に **「🎁 Trial — D-N」** というオレンジ色のバナーが表示されます。

## 2. ホーム画面

アプリのメイン画面です。追加した音声ファイルはすべてここに集まります。

![ホーム画面](images/02-home.PNG)
*▲ `images/02-home.PNG` — ホーム画面全体（上部の検索バー＋横スクロールの「最近再生」＋ファイル一覧全体）*

### 2-1. 画面構成

| 位置 | 要素 | 説明 |
|---|---|---|
| 左上 | ⚙️ 歯車 | 設定を開く |
| 右上 | ➕ 紫の丸 | ファイル追加メニュー |
| 上部 | 🔍 検索バー | ファイル名で検索 |
| 中央上 | 「最近再生」 | 最後に再生した 5 件（横スクロール） |
| 中央 | 「すべてのファイル（件数）」 | ファイル一覧全体 |

### 2-2. ファイル行の見方
各ファイルの左にあるアイコンの色で、どこから来たファイルかが分かります。

- 🟦 **ティール色の波形アイコン** — ファイルアプリから読み込んだローカルファイル
- 🟪 **紫色の音符アイコン** — ミュージックライブラリから読み込んだ曲
- 🟧 **オレンジ色のアンテナアイコン** — ポッドキャストのダウンロード

アイコンの右にはタイトル、総再生時間、前回の再生位置（あれば）、追加日が表示されます。

### 2-3. 操作

- **ファイルを 1 回タップ** → プレーヤーが開きます
- **ファイルを左にスワイプ** → 赤い 🗑 **Delete** ボタンが表示され、タップするとファイル・波形キャッシュ・字幕データがまとめて削除されます。

![スワイプでファイルを削除](images/03-home-swipe-delete.PNG)
*▲ `images/03-home-swipe-delete.PNG` — ファイル行を左にスワイプして赤い Delete ボタンが見える状態*

## 3. ファイルの追加

ホーム画面右上の **➕ ボタン** をタップしてメニューを開きます。

![追加メニュー](images/04-add-menu.PNG)
*▲ `images/04-add-menu.PNG` — ➕ をタップした後のメニュー（Add from Files / Add from Music / Subscribe to Podcast）*

### 3-1. ファイルアプリから追加
**Add from Files** → システムのファイルピッカーが開きます。mp3、m4a、wav、aac などの音声ファイルを選べます。複数同時に選択することもできます。

### 3-2. ミュージックライブラリから追加
**Add from Music** → iPhone に保存されている音楽の一覧が開きます。

- 🟢 **緑のチェックマーク** — すでに Phrabbit に追加済みの曲
- 🔒 **グレーの鍵＋「DRM」表示** — Apple Music のサブスクリプションでダウンロードした曲（著作権保護により読み込めません）。購入した曲や CD から取り込んだ曲のみ使用できます。

![ミュージックライブラリのピッカー](images/05-music-picker.PNG)
*▲ `images/05-music-picker.PNG` — ミュージックライブラリのピッカー（緑のチェックと DRM の鍵の両方が見える状態）*

曲をタップすると、自動的に m4a に変換されて Phrabbit に追加され、すぐにプレーヤーが開きます。

### 3-3. ポッドキャストを購読（プレミアム）
プレミアム中に **Subscribe to Podcast** をタップすると、Apple Podcasts の購読一覧が表示されます。
無料版ではアンテナアイコンの右上に 🔒 の鍵が付いており、タップすると購入画面に移動します。

詳しくは [10. ポッドキャスト](#10-ポッドキャスト) を参照してください。

## 4. プレーヤー：基本再生

ホーム画面でファイルをタップすると、プレーヤーがフルスクリーンで開きます。

![プレーヤー全画面](images/06-player-main.PNG)
*▲ `images/06-player-main.PNG` — プレーヤー全画面（上部の ⌄ ボタン＋波形＋A-B 情報バー＋下部コントロール＋STT テキスト領域が一目で見える）*

### 4-1. 画面を閉じる 2 つの方法

| 方法 | 説明 |
|---|---|
| 左上の ⌄ ボタン | 1 回タップでプレーヤーを下げ、ホームに戻ります |
| 上から下にスワイプ | ヘッダーをつかんで下にドラッグして閉じます |

プレーヤーを閉じても再生は続きます。画面下部に **ミニプレーヤー** が残り、タップすると再びプレーヤーが開きます。

- **波形を 1 回タップ** → その位置にジャンプ
- **波形を長押し（0.5 秒）** → A 点または B 点を設定（[5. A-B リピート](#5-a-b-リピート) を参照）

### 4-3. 下部コントロール — 上段

| ボタン | 機能 |

| **1x / 0.75x …** | 再生速度を変更。0.5 → 0.75 → 1 → 1.25 → 1.5 → 2 → 0.5 と切り替え |

![下部コントロール下段](images/09-controls-bottom-row.jpg)

| ⏪ **5 秒戻る** | 5 秒巻き戻し |

## 5. A-B リピート

中心となる機能です。好きな区間を無限に繰り返し、聞いて真似する練習ができます。

### 5-1. A-B ボタンで設定（一番簡単な方法）
下部コントロール左端の **「A-B」** カプセルを 1 回タップすると、現在の再生位置が A になります。もう一度タップするとその瞬間が B になり、すぐにリピートが始まります。

カプセルの見た目は状態によって変わります。

| 状態 | カプセルの表示 |
|---|---|
| A 設定前 | グレーの **「A-B」** |
| A のみ設定 | 紫の枠線 **「A→B」**（B 待ち） |
| A・B 両方設定 | 紫の塗りつぶし **「A-B」**（リピート中） |

![A-B カプセル — 未設定](images/10a-ab-none.PNG)
*▲ `images/10a-ab-none.PNG` — グレーの「A-B」（A 設定前）*

![A-B カプセル — B 待ち](images/10b-ab-waiting.PNG)
*▲ `images/10b-ab-waiting.PNG` — 紫の枠線「A→B」（B 待ち）*

![A-B カプセル — リピート中](images/10c-ab-active.PNG)
*▲ `images/10c-ab-active.PNG` — 紫の塗りつぶし「A-B」（A・B 両方設定、リピート中）*

A のみ設定すると、波形の上に **「👇 もう一度 A-B を押して終了点を設定」** というヒントの吹き出しが少しの間表示され、その後消えます。

### 5-2. 波形の長押しで設定（正確）
波形上の設定したい場所を **0.5 秒長押し** すると A 点が設定されます。別の場所を長押しすると B 点が設定され、リピートが始まります。

波形の上に **A ハンドル（紫）** と **B ハンドル（オレンジ）** が表示され、指でドラッグして位置を微調整できます。

![波形の A/B ハンドル](images/11-ab-handles.PNG)
*▲ `images/11-ab-handles.PNG` — 波形に A・B ハンドルが表示され、その間が紫で強調された状態*

### 5-3. A-B 情報バー
A-B 区間が設定されると、波形のすぐ下に紫の文字で情報が表示されます。

```
🔁  A: 0:12 → B: 0:18 (6.0s loop)
```

### 5-4. リピートの動作とコントロール

- **リピート回数ボタン（🔁）**：無限リピート（∞）が初期設定です。回数を設定するとその回数だけ繰り返し、自動で停止します。
- **元に戻すボタン（↶）**：すぐに A 点にジャンプします。
- **消しゴムボタン（🧹）**：A-B 区間を解除します。

## 6. ブックマーク

よく練習したい区間を保存できます。これは **プレミアム機能** で、無料版ではファイルごとに 3 つまで保存できます。

### 6-1. 保存する
1. まず A-B 区間を設定します。
2. 下部コントロールの ⭐ **星ボタン** をタップします。
3. 名前を入力して **Save** をタップします。

![ブックマーク保存ダイアログ](images/12-bookmark-save.PNG)
*▲ `images/12-bookmark-save.PNG` — ブックマークの名前入力ダイアログ*

A-B 区間を設定せずに星をタップすると、*「先に A-B 区間を設定してください。」* というアラートが表示されます。

### 6-2. リストから呼び出す
星の隣の 📋 **リストボタン** をタップすると、このファイルに保存したブックマークがシートで開きます。項目をタップすると A-B 区間が自動で設定され、すぐに再生が始まります。

![ブックマーク一覧シート](images/13-bookmark-list.PNG)
*▲ `images/13-bookmark-list.PNG` — ブックマーク一覧シート*

リスト内の項目を左にスワイプすると削除できます。

## 7. 音声テキスト変換（STT）

音声を自動で文字に起こし、聞きながら言葉を確認できます。これは **プレミアム機能** です。

### 7-1. 変換を始める
1. プレーヤー右下の **💬 字幕ボタン** をタップします。
2. 初回は **言語選択** シートが表示されます。学習している言語 — 英語、韓国語、日本語など — を選びます。
3. **Done** をタップすると変換が始まります。

![STT 言語選択シート](images/14-stt-language.PNG)
*▲ `images/14-stt-language.PNG` — STT の言語選択シート*

### 7-2. 変換の進行状況
変換が始まると、字幕エリアの上に進行バーと **「変換中…  N/合計」** のインジケーターが表示されます。

![変換中](images/15-stt-converting.PNG)
*▲ `images/15-stt-converting.PNG` — 変換中 — 進行バーと Cancel ボタンが見える状態*

- **45 秒単位** で処理するため、長い音声ほど時間がかかります（1 時間の音声で数分程度）。
- 変換の途中で ❌ ボタンで **キャンセル** できます。キャンセルすると今回の変換結果は保存されず、**既存の字幕はそのまま維持**されます。
- 変換はバックグラウンドで続きます。プレーヤーを閉じても止まりません。

### 7-3. 変換したテキストを見る
変換が終わると、下部の字幕エリアに時系列でカードが表示されます。

各カードには次の内容が含まれます：
- 開始時間（例：`0:32`）
- ソース表示（自動生成は無印、公式字幕は ✅、ユーザー編集は 🖊、手動入力は 📝）
- 認識の信頼度が低い場合は ⚠️ オレンジの警告
- 位置が目安の場合は 🕒 オレンジの **位置は目安** バッジ — テキストは認識できたものの、正確な時間情報が得られなかった区間です。タップすると近くへ移動し、✏️ で編集できます
- **本文テキスト**
- 右側の小さなボタン：🌐 翻訳 / ➕ 単語帳 / ✏️ 編集 / 🗑 削除

![字幕カード一覧](images/16-transcript-cards.PNG)
*▲ `images/16-transcript-cards.PNG` — 字幕カード一覧（複数行、現在再生中のカードはオレンジの枠線）*

### 7-4. カードの操作

- **カードをタップ** → その区間を A-B 区間として設定し、すぐにリピートを始めます。
- **🌐 翻訳**：iOS 17.4 以降の端末ではシステムの翻訳ポップアップが表示されます。
- **➕ 単語帳**：カードから単語を選んで単語帳に保存します（[9. 単語帳](#9-単語帳) を参照）。
- **✏️ 編集**：テキストを編集します。
- **🗑 削除**：カードを削除します。

現在再生中のカードは **オレンジの枠線** で強調されます。A-B 区間内に入るカードは **薄紫の背景** で表示されます。

### 7-5. 認識できなかった区間 — Gap カード
30 秒以上認識結果がなかった区間は、*「認識できませんでした。」* というグレーのカードで表示されます。**📝 Enter Manually** ボタンで自分でテキストを入力できます。

![Gap カード](images/17-gap-card.PNG)
*▲ `images/17-gap-card.PNG` — Gap カード — 「認識できませんでした」＋ Enter Manually ボタン*

### 7-6. 再実行や言語の変更
すでに変換済みのファイルで 💬 ボタンをタップするとメニューが表示されます。

- **🔄 現在の言語で再実行** — 同じ言語でもう一度変換
- **🌐 言語を変えて再実行** — 別の言語でもう一度変換
- **🗑 スクリプトを消去** — 自動生成された字幕のみ削除（手動編集分は残ります）

![STT 管理メニュー](images/18-stt-manage-menu.PNG)
*▲ `images/18-stt-manage-menu.PNG` — STT 管理メニュー（3 つの選択肢）*

> 💡 **ヒント**：認識精度が低すぎると「音声認識の精度が低い可能性があります」というアラートが表示されます。これは音声の実際の言語と選んだ言語が違う場合によく起こります。**Change Language** ボタンで言語を切り替えてもう一度試してください。

## 8. 字幕の編集と追加

### 8-1. 字幕を手動で追加
字幕エリア上部の **➕ Add Segment** ボタンをタップすると、現在の再生位置を基準に新しい区間を作成します。シートにテキストを入力し、**Add** をタップするとカードが追加されます。

![Add Segment シート](images/19-add-segment.PNG)
*▲ `images/19-add-segment.PNG` — Add Segment シート（時間範囲＋テキスト入力＋▶ プレビューボタン）*

シート内の **▶ プレビューボタン** で、その区間だけを少し試聴できます。

### 8-2. 字幕を編集
カードの ✏️ ボタンをタップすると編集シートが開き、**字幕のテキストを編集** できます。上部の ▶ **Loop ボタン** で区間を繰り返し聞きながら書き取ることができ、STT で生成された字幕を編集した場合は、下に **Original** と **Reset** ボタンが表示され、いつでも元に戻せます。（字幕の時間範囲はここでは変更できません。上部に参考表示されるだけです。）

![字幕編集シート](images/19-edit-segment.PNG)
*▲ `images/19-edit-segment.PNG` — 字幕編集シート（テキスト編集＋Loop プレビュー）*

## 9. 単語帳

字幕カードから保存した単語を集める場所です。これは **プレミアム機能** です。

![単語帳リストモード](images/20-wordbook-list.PNG)
*▲ `images/20-wordbook-list.PNG` — 単語帳タブ — リストモード*

### 9-1. 単語を保存する
プレーヤーの字幕カードで ➕ ボタンをタップ → **単語選択** 画面で：

- カード本文から自動抽出された単語がカプセルで表示されます。タップして選ぶか、**Custom Input** に自分で入力します。
- メモを追加できます（任意）。
- **Save** をタップすると単語帳に保存されます。

![単語保存画面](images/20-wordbook-save.PNG)
*▲ `images/20-wordbook-save.PNG` — 字幕カードから単語を選んで保存する画面*

### 9-2. 2 つの表示モード
画面上部のセグメントコントロールでモードを切り替えます。

- **リスト** — 単語、その文の文脈、ソースファイル、追加日、メモを一目で確認できます。項目を左にスワイプすると削除できます。
- **フラッシュカード** — カードをタップすると表と裏が反転します（表：単語、裏：文＋メモ）。左右にスワイプすると次のカードに移動します。

![フラッシュカード — 表](images/21a-wordbook-flashcard.PNG)

## 10. ポッドキャスト

Apple Podcasts で購読しているポッドキャストを学習教材として取り込みます。これは **プレミアム機能** です。

### 10-1. 開く
ホーム画面の ➕ メニュー → **Subscribe to Podcast**。

### 10-2. Apple Podcasts から取り込む
初回は **「Load from Apple Podcasts」** ボタンが表示されます。タップするとメディアライブラリの許可を求められ、許可すると購読中のポッドキャストが横スクロールのカードで表示されます。

![ポッドキャスト画面](images/22-podcast.PNG)
*▲ `images/22-podcast.PNG` — ポッドキャスト画面 — Apple Podcasts の購読カードが横スクロール*

チャンネルカードをタップすると、そのエピソード一覧がシートで開きます。

![エピソード一覧シート](images/23a-podcast-episodes.PNG)
*▲ `images/23a-podcast-episodes.PNG` — エピソード一覧シート（公式字幕のない通常のエピソード）*

各エピソードについて：
- タイトル、公開日、長さ
- **Script** バッジ（紫）— 公式の時間同期字幕を提供するエピソード
- **Text only** バッジ（グレー）— テキストのみのエピソード
- バッジなし — 字幕を提供しないエピソード
- 右側のステータス：
  - ⬇ ダウンロードアイコン — 未ダウンロード
  - 🟣 進行リング — ダウンロード中
  - ✅ 緑のチェック — ダウンロード完了

チェックの付いたエピソードをタップすると、すぐにプレーヤーが開きます。

![Script バッジ付きエピソード](images/23b-podcast-episodes-script.PNG)
*▲ `images/23b-podcast-episodes-script.PNG` — 紫の **Script** バッジが付いたエピソード一覧*

> 💡 **Script エピソードをおすすめする理由**：**Script** バッジの付いたエピソードは、iOS の音声認識（STT）で字幕を作るのではなく、**ポッドキャストが提供する公式字幕を直接取り込みます**。人の手で書かれた正確な字幕なので、STT に比べて誤字や誤認識がはるかに少なく、変換を待つ必要もありません。学習教材を選ぶときは、Script エピソードを優先して選ぶことをおすすめします。

### 10-3. RSS URL で直接取り込む
画面下部の **「Add via RSS URL (advanced)」** を展開すると、RSS アドレスを直接入力できます。Apple Podcasts にないポッドキャストを取り込むときに使います。

### 10-4. モバイルデータ通信について
初期設定は **Wi-Fi でのみダウンロード** です。設定でモバイルデータ通信のダウンロードを許可している場合は、大きなエピソードをダウンロードする前に確認アラートが表示されます。

## 11. 設定

ホーム画面左上の ⚙️ 歯車アイコンで開きます。

![設定画面](images/24-settings.jpg)

### 11-1. ポッドキャストのダウンロード
- **Wi-Fi でのみダウンロード** — オンにすると Wi-Fi 接続時のみダウンロードします（初期設定：オン）。この設定は STT（音声→テキスト）の時間情報を復元するためのサーバー認識にも適用されます — オンなら Wi-Fi のみ、オフならモバイルデータ通信でもサーバー認識を使います。
- **モバイル通信前に確認** — Wi-Fi 限定がオフのときに表示され、モバイル通信でダウンロードする前に毎回確認するかどうかを決めます。

## 12. 無料版とプレミアム

### ずっと無料
- 波形表示＋A-B リピート
- 再生速度の調整
- ローカルファイルの取り込み（ファイルアプリ）
- ミュージックライブラリからの取り込み
- ブックマーク（ファイルごとに 3 つまで）

### プレミアム
- 音声テキスト変換（STT）
- リアルタイム字幕 ↔ 波形の同期
- 単語帳
- ブックマーク無制限
- ポッドキャストのダウンロード

### 支払いの仕組み
**3 日間の無料トライアル → 買い切り。** 一度購入すれば一生使えます — サブスクリプションではありません。端末を変えても、同じ Apple ID で **「Restore Purchases」** すれば引き継げます。

## 13. よくある質問

**Q. Apple Music で聞いた曲を取り込めないのはなぜですか？**
A. Apple Music のサブスクリプションでダウンロードした曲は著作権保護（DRM）でロックされており、他のアプリでは使えません。iTunes Store で購入した曲や CD から取り込んだ曲のみ使用できます。

**Q. 音声認識が不正確すぎます。**
A. 2 つを確認してください：① 音声の実際の言語と選んだ言語が一致しているか、② BGM やノイズが多くないか。認識が難しすぎる区間は、➕ Add Segment や Enter Manually で自分で入力してください。

**Q. 変換中に別のファイルをタップするとどうなりますか？**
A. *「... に切り替えますか？変換が中断されます。途中までの結果は保存されます。」* という確認が表示されます。そこまで完了した部分は保持されます。

**Q. 私のデータはインターネットに送信されますか？**
A. 音声認識は基本的に **端末内（オンデバイス）** で処理されます。ただしオンデバイス認識が一部区間の正確な時間情報を作れない場合は、**Wi-Fi 接続時（設定でモバイルデータ通信を許可している場合はインターネット接続時）に、その音声区間のみ Apple の音声認識サーバーへ送信**して位置を復元します。第三者のサーバーは使用せず、単語帳・ブックマーク・字幕はすべて端末内に保存されます。インターネットに接続していない場合は、すべての認識が端末内のみで処理されます。

**Q. 端末を変えるとデータはどうなりますか？**
A. 現在のバージョン（v1.0）はデータを端末内のみに保存します。新しい端末にアプリをインストールし、同じ Apple ID で **Restore Purchases** すればプレミアムは解除されますが、単語帳・ブックマーク・字幕は引き継がれません。
