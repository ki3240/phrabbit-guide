# Phrabbit 用户指南

🌐 [한국어](ko.md) · [English](en.md) · [日本語](ja.md) · **简体中文** · [繁體中文](zh-Hant.md)

## 1. 开始使用

### 1-1. 首次启动 — 引导页
首次打开应用时，会出现三张介绍幻灯片。

1. **用波形精确选择片段** — 把音频以图形方式显示出来，让你精准指定想听的部分。
2. **在 A、B 两点之间无限循环** — 设置起点（A）和终点（B），就只会自动重复那一段。
3. **实时语音转文字** — 你正在听的音频会被自动转成文字，方便边听边看。

用右上角的 **「Skip」** 按钮跳过，或在最后一张幻灯片用 **「Get Started」** 按钮开始。

![引导页幻灯片](images/01-onboarding.PNG)
*▲ `images/01-onboarding.PNG` — 第一张引导页幻灯片（含波形动画）*

### 1-2. 免费试用
首次启动会自动开始 **3 天免费试用**。这三天里你可以自由体验所有功能。试用期间，播放器顶部会显示一条写着 **「🎁 Trial — D-N」** 的橙色横幅。

## 2. 主屏幕

这是应用的主界面。你添加的所有音频文件都会汇集在这里。

![主屏幕](images/02-home.PNG)
*▲ `images/02-home.PNG` — 主屏幕全貌（顶部搜索栏 + 横向滚动的「最近播放」+ 完整文件列表）*

### 2-1. 界面布局

| 位置 | 元素 | 说明 |
|---|---|---|
| 左上 | ⚙️ 齿轮 | 打开设置 |
| 右上 | ➕ 紫色圆圈 | 文件添加菜单 |
| 顶部 | 🔍 搜索栏 | 按文件名搜索 |
| 中上 | 「最近播放」 | 最近播放的 5 个文件（横向滚动） |
| 中部 | 「所有文件（数量）」 | 完整文件列表 |

### 2-2. 看懂文件行
每个文件左侧图标的颜色，告诉你这个文件来自哪里。

- 🟦 **青绿色波形图标** — 从「文件」应用导入的本地文件

- **轻点文件一次** → 打开播放器
- **向左滑动文件** → 出现红色 🗑 **Delete** 按钮，点击即可一并删除文件、波形缓存和字幕数据。

## 3. 导入文件

点击主屏幕右上角的 **➕ 按钮** 打开菜单。

![添加菜单](images/04-add-menu.PNG)
*▲ `images/04-add-menu.PNG` — 点击 ➕ 后的菜单（Add from Files / Add from Music / Subscribe to Podcast）*

### 3-1. 从「文件」应用添加
**Add from Files** → 打开系统文件选择器。可以选择 mp3、m4a、wav、aac 等音频文件，也可以一次选多个。

### 3-2. 从音乐库添加
**Add from Music** → 打开存储在 iPhone 上的音乐列表。

- 🟢 **绿色对勾** — 已经添加到 Phrabbit 的曲目
- 🔒 **灰色锁 +「DRM」标记** — 通过 Apple Music 订阅下载的曲目（受版权保护，无法导入）。只有你单独购买或从 CD 导入的曲目才能使用。

![音乐库选择器](images/05-music-picker.PNG)
*▲ `images/05-music-picker.PNG` — 音乐库选择器（同时显示绿色对勾和 DRM 锁）*

点击曲目后，它会被自动转换为 m4a 并添加到 Phrabbit，随即打开播放器。

### 3-3. 订阅播客（高级版）
在高级版有效期间，点击 **Subscribe to Podcast** 会显示你在 Apple Podcasts 的订阅列表。
免费版中，天线图标右上角会有一个 🔒 锁，点击会跳转到购买页面。

详情请见 [10. 播客](#10-播客)。

## 4. 播放器：基本播放

在主屏幕点击文件，播放器会全屏打开。

![播放器全屏](images/06-player-main.PNG)
*▲ `images/06-player-main.PNG` — 播放器全屏（顶部 ⌄ 按钮 + 波形 + A-B 信息栏 + 底部控件 + STT 文字区域，一目了然）*

### 4-1. 关闭界面的两种方式

| 方式 | 说明 |
|---|---|
| 左上角 ⌄ 按钮 | 点一下即可收起播放器并返回主屏幕 |
| 从顶部向下滑 | 抓住标题栏向下拖动即可关闭 |

关闭播放器后播放仍会继续。屏幕底部会保留一个 **迷你播放器**，点击它即可重新打开播放器。

![迷你播放器](images/07-mini-player.PNG)

### 4-2. 波形区域
在时间显示（例如 `0:00`）旁边，波形以 5 秒为单位横向展开，播放时一条黑色竖线（播放头）会随之移动。屏幕也会自动滚动以跟随播放位置。

- **轻点波形一次** → 跳到该位置
- **长按波形（0.5 秒）** → 设置 A 点或 B 点（见 [5. A-B 循环](#5-a-b-循环)）

### 4-3. 底部控件 — 上排

![底部控件上排](images/08-controls-top-row.jpg)

| 按钮 | 功能 |
|---|---|
| **A-B** 胶囊 | 点击依次设置 A、B 两点 |

| 🔁 **循环次数** | 每点一次循环切换 ∞ → 1 → 2 → 3 → 5 → 10 → ∞ |
| **1x / 0.75x …** | 改变播放速度。循环 0.5 → 0.75 → 1 → 1.25 → 1.5 → 2 → 0.5 |

## 5. A-B 循环

核心功能：把任意片段无限循环，让你边听边跟读。

### 5-1. 用 A-B 按钮设置（最简单的方式）
点击底部控件最左侧的 **「A-B」** 胶囊一次，当前播放位置就成为 A。再点一次，那一刻就成为 B，循环立即开始。

胶囊的外观会随状态变化。

| 状态 | 胶囊显示 |
|---|---|
| 设置 A 之前 | 灰色 **「A-B」** |
| 仅设置了 A | 紫色描边 **「A→B」**（等待 B） |
| A、B 都已设置 | 紫色填充 **「A-B」**（循环中） |

![A-B 胶囊 — 未设置](images/10a-ab-none.PNG)
*▲ `images/10a-ab-none.PNG` — 灰色「A-B」（设置 A 之前）*

![A-B 胶囊 — 等待 B](images/10b-ab-waiting.PNG)
*▲ `images/10b-ab-waiting.PNG` — 紫色描边「A→B」（等待 B）*

![A-B 胶囊 — 循环中](images/10c-ab-active.PNG)
*▲ `images/10c-ab-active.PNG` — 紫色填充「A-B」（A、B 都已设置，循环中）*

仅设置了 A 时，波形上方会短暂出现一个提示气泡 **「👇 再点一次 A-B 设置终点」**，随后淡出。

### 5-2. 长按波形设置（精确）
在波形上想要的位置 **长按 0.5 秒** 即可设置 A 点。长按另一处可设置 B 点并开始循环。

波形上方会出现 **A 手柄（紫色）** 和 **B 手柄（橙色）**，你可以用手指拖动它们来微调位置。

![波形 A/B 手柄](images/11-ab-handles.PNG)
*▲ `images/11-ab-handles.PNG` — 波形上显示 A、B 手柄，两者之间的区段以紫色高亮*

### 5-3. A-B 信息栏
设置好 A-B 区间后，信息会以紫色文字显示在波形正下方。

```
🔁  A: 0:12 → B: 0:18 (6.0s loop)
```

### 5-4. 循环行为与控件

- **循环次数按钮（🔁）**：默认是无限循环（∞）。设置一个次数后，它会重复那么多次然后自动停止。
- **撤销按钮（↶）**：立即跳到 A 点。
- **橡皮擦按钮（🧹）**：清除 A-B 区间。

## 6. 书签

保存你想经常练习的片段。这是 **高级版功能**；免费用户每个文件最多保存 3 个。

### 6-1. 保存
1. 先设置好 A-B 区间。
2. 点击底部控件的 ⭐ **星标按钮**。
3. 输入名称并点击 **Save**。

![书签保存对话框](images/12-bookmark-save.PNG)
*▲ `images/12-bookmark-save.PNG` — 书签命名对话框*

如果在未设置 A-B 区间时点击星标，会出现 *「请先设置 A-B 区间。」* 的提示。

### 6-2. 从列表中载入
点击星标旁边的 📋 **列表按钮**，会以工作表打开此文件已保存的书签。点击某一项，其 A-B 区间会自动设置并立即开始播放。

![书签列表工作表](images/13-bookmark-list.PNG)
*▲ `images/13-bookmark-list.PNG` — 书签列表工作表*

在列表中向左滑动某一项即可删除。

## 7. 语音转文字（STT）

自动把音频转成文字，让你边听边确认词句。这是 **高级版功能**。

### 7-1. 开始转换
1. 点击播放器右下角的 **💬 字幕按钮**。
2. 首次使用时会出现 **语言选择** 工作表。选择你正在学习的语言 — 英语、韩语、日语等。
3. 点击 **Done** 开始转换。

![STT 语言选择工作表](images/14-stt-language.PNG)
*▲ `images/14-stt-language.PNG` — STT 语言选择工作表*

### 7-2. 转换进度
转换开始后，字幕区域上方会出现进度条和 **「转换中…  N/总数」** 指示。

![转换中](images/15-stt-converting.PNG)
*▲ `images/15-stt-converting.PNG` — 转换中 — 可见进度条和 Cancel 按钮*

- 由于它 **以 45 秒为单位** 处理，音频越长用时越久（一小时的音频大约几分钟）。
- 你可以用 ❌ 按钮在转换途中 **取消**。取消后本次转换结果不会保存，**原有字幕保持不变**。
- 转换会在后台继续运行。即使关闭播放器也不会停止。

### 7-3. 查看转换后的文字
转换完成后，字幕区域底部会按时间顺序出现卡片。

每张卡片包含：
- 开始时间（例如 `0:32`）
- 来源标记（自动生成无标记，✅ 表示官方字幕，🖊 表示用户编辑，📝 表示手动输入）
- 若识别置信度低，会有 ⚠️ 橙色警告
- 若位置为大致估算，会有 🕒 橙色 **大致位置** 徽章 — 已识别出文本但无法确定精确时间的片段。点按可跳到附近，并可用 ✏️ 编辑
- **正文文字**
- 右侧小按钮：🌐 翻译 / ➕ 单词本 / ✏️ 编辑 / 🗑 删除

![字幕卡片列表](images/16-transcript-cards.PNG)
*▲ `images/16-transcript-cards.PNG` — 字幕卡片列表（多行，当前正在播放的卡片以橙色边框标出）*

### 7-4. 卡片操作

- **点击卡片** → 把该片段设为 A-B 区间并立即开始循环。
- **🌐 翻译**：会弹出系统翻译窗口。
- **➕ 单词本**：从卡片中选一个单词存入单词本（见 [9. 单词本](#9-单词本)）。
- **✏️ 编辑**：编辑文字。
- **🗑 删除**：删除卡片。

当前正在播放的卡片会以 **橙色边框** 高亮。落在 A-B 区间内的卡片会显示 **浅紫色背景**。

### 7-5. 无法识别的片段 — Gap 卡片
连续 30 秒或更久没有识别结果的片段，会显示为一张写着 *「无法识别。」* 的灰色卡片。你可以用 **📝 Enter Manually** 按钮自己输入文字。

![Gap 卡片](images/17-gap-card.PNG)
*▲ `images/17-gap-card.PNG` — Gap 卡片 —「无法识别」+ Enter Manually 按钮*

### 7-6. 重新运行或更改语言
在已转换的文件上点击 💬 按钮会弹出菜单。

- **🔄 用当前语言重新运行** — 用同一语言再转换一次
- **🌐 更改语言并重新运行** — 用另一种语言再转换一次
- **🗑 清除脚本** — 只删除自动生成的字幕（你的手动编辑会保留）

![STT 管理菜单](images/18-stt-manage-menu.PNG)
*▲ `images/18-stt-manage-menu.PNG` — STT 管理菜单（3 个选项）*

> 💡 **提示**：如果识别准确率太低，会出现「语音识别准确率可能较低」的提示。这通常发生在音频的实际语言与你所选语言不同的时候。请用 **Change Language** 按钮切换语言再试一次。

## 8. 编辑与添加字幕

### 8-1. 手动添加字幕
点击字幕区域顶部的 **➕ Add Segment** 按钮，会根据当前播放位置创建一个新片段。在工作表中输入文字并点击 **Add** 即可添加卡片。

![Add Segment 工作表](images/19-add-segment.PNG)
*▲ `images/19-add-segment.PNG` — Add Segment 工作表（时间范围 + 文字输入 + ▶ 预听按钮）*

工作表中的 **▶ 预听按钮** 让你可以短暂试听那一段。

### 8-2. 编辑字幕
点击卡片上的 ✏️ 按钮打开编辑工作表，可以 **编辑字幕的文字**。用顶部的 ▶ **Loop 按钮** 可以一边循环听该片段一边听写；如果你编辑的是 STT 生成的字幕，下方还会出现 **Original** 和 **Reset** 按钮，让你随时还原成原文。（字幕的时间范围在这里无法更改，仅在顶部显示以供参考。）

![字幕编辑工作表](images/19-edit-segment.PNG)

## 10. 播客

把你在 Apple Podcasts 订阅的播客作为学习素材引入。这是 **高级版功能**。

### 10-1. 打开
主屏幕 ➕ 菜单 → **Subscribe to Podcast**。

### 10-2. 从 Apple Podcasts 导入
首次使用时会看到 **「Load from Apple Podcasts」** 按钮。点击它会请求媒体库权限，授权后，你订阅的播客会以横向滚动的卡片显示。

![播客界面](images/22-podcast.PNG)
*▲ `images/22-podcast.PNG` — 播客界面 — Apple Podcasts 订阅卡片横向滚动*

点击频道卡片，会以工作表打开它的剧集列表。

![剧集列表工作表](images/23a-podcast-episodes.PNG)
*▲ `images/23a-podcast-episodes.PNG` — 剧集列表工作表（没有官方字幕的普通剧集）*

每个剧集会显示：
- 标题、发布日期、时长
- **Script** 徽章（紫色）— 提供官方时间同步字幕的剧集
- **Text only** 徽章（灰色）— 仅有文字的剧集
- 无徽章 — 不提供字幕的剧集
- 右侧状态：
  - ⬇ 下载图标 — 尚未下载
  - 🟣 进度环 — 下载中
  - ✅ 绿色对勾 — 下载完成

### 10-3. 通过 RSS URL 直接导入
展开屏幕底部附近的 **「Add via RSS URL (advanced)」**，即可直接输入 RSS 地址。用它来导入 Apple Podcasts 中没有的播客。

### 10-4. 关于蜂窝数据
默认设置为 **仅在 Wi-Fi 下下载**。如果你在设置里允许了蜂窝数据下载，那么在下载较大的剧集前会出现确认提示。

## 13. 常见问题

**Q. 为什么我在 Apple Music 上听的曲目无法导入？**
A. 通过 Apple Music 订阅下载的曲目受版权保护（DRM）锁定，无法在其他应用中使用。只有从 iTunes Store 购买或从 CD 导入的曲目才能使用。

**Q. 语音识别太不准了。**
A. 检查两点：① 音频的实际语言是否与你所选语言一致，② 是否有大量背景音乐或噪音。对于实在难以识别的片段，直接用 ➕ Add Segment 或 Enter Manually 自己输入即可。

**Q. 转换过程中点了另一个文件会怎样？**
A. 会出现 *「切换到 ...？转换将被中断。已完成部分会被保存。」* 的确认。已经完成的部分会被保留。

**Q. 我的数据会通过互联网发送吗？**
A. 语音识别默认在**设备端（本地）**进行。不过，当设备端识别无法确定某些片段的精确时间时，Phrabbit 会**仅在 Wi-Fi 下（若你在设置中允许了蜂窝数据，则为任意联网状态）把那一小段音频发送到 Apple 的语音识别服务器**以恢复其位置。不使用任何第三方服务器，你的单词本、书签和字幕都存储在你的设备上。在没有联网时，所有识别都仅在设备端进行。

**Q. 更换设备后我的数据会怎样？**
A. 当前版本（v1.0）只把数据存储在设备上。在新设备上安装应用并用同一 Apple ID 执行 **Restore Purchases** 可以解锁高级版，但单词本、书签和字幕不会转移。
