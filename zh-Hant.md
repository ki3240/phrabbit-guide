# Phrabbit 使用指南

🌐 [한국어](ko.md) · [English](en.md) · [日本語](ja.md) · [简体中文](zh-Hans.md) · **繁體中文**

一款用 A/B 區間循環音訊來練習外語的 iOS 應用程式。載入音訊檔案，在波形上自由循環任意片段，並自動把語音轉成文字，讓你邊聽邊看。

## 1. 開始使用

### 1-1. 首次啟動 — 導覽頁
首次開啟應用程式時，會出現三張介紹投影片。

1. **用波形精確選擇片段** — 把音訊以圖形方式顯示出來，讓你精準指定想聽的部分。
2. **在 A、B 兩點之間無限循環** — 設定起點（A）和終點（B），就只會自動重複那一段。
3. **即時語音轉文字** — 你正在聽的音訊會被自動轉成文字，方便邊聽邊看。

用右上角的 **「Skip」** 按鈕跳過，或在最後一張投影片用 **「Get Started」** 按鈕開始。

![導覽頁投影片](images/01-onboarding.PNG)
*▲ `images/01-onboarding.PNG` — 第一張導覽頁投影片（含波形動畫）*

### 1-2. 免費試用
首次啟動會自動開始 **3 天免費試用**。這三天裡你可以自由體驗所有功能。試用期間，播放器頂部會顯示一條寫著 **「🎁 Trial — D-N」** 的橘色橫幅。

## 2. 主畫面

這是應用程式的主介面。你新增的所有音訊檔案都會匯集在這裡。

![主畫面](images/02-home.PNG)
*▲ `images/02-home.PNG` — 主畫面全貌（頂部搜尋列 + 橫向捲動的「最近播放」+ 完整檔案清單）*

### 2-1. 介面配置

| 位置 | 元素 | 說明 |
|---|---|---|
| 左上 | ⚙️ 齒輪 | 開啟設定 |
| 右上 | ➕ 紫色圓圈 | 檔案新增選單 |
| 頂部 | 🔍 搜尋列 | 依檔案名稱搜尋 |
| 中上 | 「最近播放」 | 最近播放的 5 個檔案（橫向捲動） |
| 中部 | 「所有檔案（數量）」 | 完整檔案清單 |

### 2-2. 看懂檔案列
每個檔案左側圖示的顏色，告訴你這個檔案來自哪裡。

- 🟦 **青綠色波形圖示** — 從「檔案」應用程式匯入的本機檔案
- 🟪 **紫色音符圖示** — 從音樂資料庫匯入的曲目
- 🟧 **橘色天線圖示** — Podcast 下載

圖示右側會顯示標題、總時長、上次播放位置（如果有）和新增日期。

### 2-3. 操作

- **輕點檔案一次** → 開啟播放器
- **向左滑動檔案** → 出現紅色 🗑 **Delete** 按鈕，點擊即可一併刪除檔案、波形快取和字幕資料。

![滑動刪除檔案](images/03-home-swipe-delete.PNG)
*▲ `images/03-home-swipe-delete.PNG` — 檔案列向左滑動後露出紅色 Delete 按鈕*

## 3. 匯入檔案

點擊主畫面右上角的 **➕ 按鈕** 開啟選單。

![新增選單](images/04-add-menu.PNG)
*▲ `images/04-add-menu.PNG` — 點擊 ➕ 後的選單（Add from Files / Add from Music / Subscribe to Podcast）*

### 3-1. 從「檔案」應用程式新增
**Add from Files** → 開啟系統檔案選擇器。可以選擇 mp3、m4a、wav、aac 等音訊檔案，也可以一次選多個。

### 3-2. 從音樂資料庫新增
**Add from Music** → 開啟儲存在 iPhone 上的音樂清單。

- 🟢 **綠色勾選** — 已經新增到 Phrabbit 的曲目
- 🔒 **灰色鎖 +「DRM」標記** — 透過 Apple Music 訂閱下載的曲目（受版權保護，無法匯入）。只有你單獨購買或從 CD 匯入的曲目才能使用。

![音樂資料庫選擇器](images/05-music-picker.PNG)
*▲ `images/05-music-picker.PNG` — 音樂資料庫選擇器（同時顯示綠色勾選和 DRM 鎖）*

點擊曲目後，它會被自動轉換為 m4a 並新增到 Phrabbit，隨即開啟播放器。

### 3-3. 訂閱 Podcast（進階版）
在進階版有效期間，點擊 **Subscribe to Podcast** 會顯示你在 Apple Podcasts 的訂閱清單。
免費版中，天線圖示右上角會有一個 🔒 鎖，點擊會跳轉到購買頁面。

詳情請見 [10. Podcast](#10-podcast)。

## 4. 播放器：基本播放

在主畫面點擊檔案，播放器會全螢幕開啟。

![播放器全螢幕](images/06-player-main.PNG)
*▲ `images/06-player-main.PNG` — 播放器全螢幕（頂部 ⌄ 按鈕 + 波形 + A-B 資訊列 + 底部控制項 + STT 文字區域，一目了然）*

### 4-1. 關閉畫面的兩種方式

| 方式 | 說明 |
|---|---|
| 左上角 ⌄ 按鈕 | 點一下即可收起播放器並返回主畫面 |
| 從頂部向下滑 | 抓住標題列向下拖動即可關閉 |

關閉播放器後播放仍會繼續。畫面底部會保留一個 **迷你播放器**，點擊它即可重新開啟播放器。

![迷你播放器](images/07-mini-player.PNG)
*▲ `images/07-mini-player.PNG` — 浮在主畫面底部的迷你播放器*

### 4-2. 波形區域
在時間顯示（例如 `0:00`）旁邊，波形以 5 秒為單位橫向展開，播放時一條黑色直線（播放頭）會隨之移動。畫面也會自動捲動以跟隨播放位置。

- **輕點波形一次** → 跳到該位置
- **長按波形（0.5 秒）** → 設定 A 點或 B 點（見 [5. A-B 循環](#5-a-b-循環)）

### 4-3. 底部控制項 — 上排

![底部控制項上排](images/08-controls-top-row.jpg)
*▲ `images/08-controls-top-row.jpg` — 僅底部控制項的上排（從左到右：A-B / 復原 / 橡皮擦 / 分隔線 / 循環次數 / 播放速度）*

| 按鈕 | 功能 |
|---|---|
| **A-B** 膠囊 | 點擊依序設定 A、B 兩點 |
| ↶ **復原** | 跳到目前 A 點 |
| 🧹 **橡皮擦** | 清除 A-B 區間 |
| 🔁 **循環次數** | 每點一次循環切換 ∞ → 1 → 2 → 3 → 5 → 10 → ∞ |
| **1x / 0.75x …** | 改變播放速度。循環 0.5 → 0.75 → 1 → 1.25 → 1.5 → 2 → 0.5 |

### 4-4. 底部控制項 — 下排

![底部控制項下排](images/09-controls-bottom-row.jpg)
*▲ `images/09-controls-bottom-row.jpg` — 僅底部控制項的下排（左：星號和清單 / 中：後退 5 秒 ▶ 前進 5 秒 / 右：字幕麥克風按鈕）*

| 按鈕 | 功能 |
|---|---|
| ⭐ **星號** | 把目前 A-B 區間儲存為書籤（進階版） |
| 📋 **清單** | 開啟已儲存的書籤清單（進階版） |
| ⏪ **後退 5 秒** | 倒退 5 秒 |
| ⏯ **大播放按鈕** | 播放 / 暫停 |
| ⏩ **前進 5 秒** | 快轉 5 秒 |
| 💬 **字幕 / 麥克風** | 語音轉文字（進階版） |

## 5. A-B 循環

核心功能：把任意片段無限循環，讓你邊聽邊跟讀。

### 5-1. 用 A-B 按鈕設定（最簡單的方式）
點擊底部控制項最左側的 **「A-B」** 膠囊一次，目前播放位置就成為 A。再點一次，那一刻就成為 B，循環立即開始。

膠囊的外觀會隨狀態變化。

| 狀態 | 膠囊顯示 |
|---|---|
| 設定 A 之前 | 灰色 **「A-B」** |
| 僅設定了 A | 紫色外框 **「A→B」**（等待 B） |
| A、B 都已設定 | 紫色填滿 **「A-B」**（循環中） |

![A-B 膠囊 — 未設定](images/10a-ab-none.PNG)
*▲ `images/10a-ab-none.PNG` — 灰色「A-B」（設定 A 之前）*

![A-B 膠囊 — 等待 B](images/10b-ab-waiting.PNG)
*▲ `images/10b-ab-waiting.PNG` — 紫色外框「A→B」（等待 B）*

![A-B 膠囊 — 循環中](images/10c-ab-active.PNG)
*▲ `images/10c-ab-active.PNG` — 紫色填滿「A-B」（A、B 都已設定，循環中）*

僅設定了 A 時，波形上方會短暫出現一個提示氣泡 **「👇 再點一次 A-B 設定終點」**，隨後淡出。

### 5-2. 長按波形設定（精確）
在波形上想要的位置 **長按 0.5 秒** 即可設定 A 點。長按另一處可設定 B 點並開始循環。

波形上方會出現 **A 控制點（紫色）** 和 **B 控制點（橘色）**，你可以用手指拖動它們來微調位置。

![波形 A/B 控制點](images/11-ab-handles.PNG)
*▲ `images/11-ab-handles.PNG` — 波形上顯示 A、B 控制點，兩者之間的區段以紫色標示*

### 5-3. A-B 資訊列
設定好 A-B 區間後，資訊會以紫色文字顯示在波形正下方。

```
🔁  A: 0:12 → B: 0:18 (6.0s loop)
```

### 5-4. 循環行為與控制項

- **循環次數按鈕（🔁）**：預設是無限循環（∞）。設定一個次數後，它會重複那麼多次然後自動停止。
- **復原按鈕（↶）**：立即跳到 A 點。
- **橡皮擦按鈕（🧹）**：清除 A-B 區間。

## 6. 書籤

儲存你想經常練習的片段。這是 **進階版功能**；免費使用者每個檔案最多儲存 3 個。

### 6-1. 儲存
1. 先設定好 A-B 區間。
2. 點擊底部控制項的 ⭐ **星號按鈕**。
3. 輸入名稱並點擊 **Save**。

![書籤儲存對話框](images/12-bookmark-save.PNG)
*▲ `images/12-bookmark-save.PNG` — 書籤命名對話框*

如果在未設定 A-B 區間時點擊星號，會出現 *「請先設定 A-B 區間。」* 的提示。

### 6-2. 從清單中載入
點擊星號旁邊的 📋 **清單按鈕**，會以工作表開啟此檔案已儲存的書籤。點擊某一項，其 A-B 區間會自動設定並立即開始播放。

![書籤清單工作表](images/13-bookmark-list.PNG)
*▲ `images/13-bookmark-list.PNG` — 書籤清單工作表*

在清單中向左滑動某一項即可刪除。

## 7. 語音轉文字（STT）

自動把音訊轉成文字，讓你邊聽邊確認詞句。這是 **進階版功能**。

### 7-1. 開始轉換
1. 點擊播放器右下角的 **💬 字幕按鈕**。
2. 首次使用時會出現 **語言選擇** 工作表。選擇你正在學習的語言 — 英文、韓文、日文等。
3. 點擊 **Done** 開始轉換。

![STT 語言選擇工作表](images/14-stt-language.PNG)
*▲ `images/14-stt-language.PNG` — STT 語言選擇工作表*

### 7-2. 轉換進度
轉換開始後，字幕區域上方會出現進度條和 **「轉換中…  N/總數」** 指示。

![轉換中](images/15-stt-converting.PNG)
*▲ `images/15-stt-converting.PNG` — 轉換中 — 可見進度條和 Cancel 按鈕*

- 由於它 **以 45 秒為單位** 處理，音訊越長用時越久（一小時的音訊大約幾分鐘）。
- 你可以用 ❌ 按鈕在轉換途中 **取消**。已經完成的部分會照原樣儲存。
- 轉換會在背景繼續執行。即使關閉播放器也不會停止。

### 7-3. 查看轉換後的文字
轉換完成後，字幕區域底部會依時間順序出現卡片。

每張卡片包含：
- 開始時間（例如 `0:32`）
- 來源標記（自動產生無標記，✅ 表示官方字幕，🖊 表示使用者編輯，📝 表示手動輸入）
- 若辨識信賴度低，會有 ⚠️ 橘色警告
- **正文文字**
- 右側小按鈕：🌐 翻譯 / ➕ 單字本 / ✏️ 編輯 / 🗑 刪除

![字幕卡片清單](images/16-transcript-cards.PNG)
*▲ `images/16-transcript-cards.PNG` — 字幕卡片清單（多列，目前正在播放的卡片以橘色外框標出）*

### 7-4. 卡片操作

- **點擊卡片** → 把該片段設為 A-B 區間並立即開始循環。
- **🌐 翻譯**：在 iOS 17.4 以上的裝置上會彈出系統翻譯視窗。
- **➕ 單字本**：從卡片中選一個單字存入單字本（見 [9. 單字本](#9-單字本)）。
- **✏️ 編輯**：編輯文字。
- **🗑 刪除**：刪除卡片。

目前正在播放的卡片會以 **橘色外框** 標示。落在 A-B 區間內的卡片會顯示 **淺紫色背景**。

### 7-5. 無法辨識的片段 — Gap 卡片
連續 30 秒或更久沒有辨識結果的片段，會顯示為一張寫著 *「無法辨識。」* 的灰色卡片。你可以用 **📝 Enter Manually** 按鈕自己輸入文字。

![Gap 卡片](images/17-gap-card.PNG)
*▲ `images/17-gap-card.PNG` — Gap 卡片 —「無法辨識」+ Enter Manually 按鈕*

### 7-6. 重新執行或變更語言
在已轉換的檔案上點擊 💬 按鈕會彈出選單。

- **🔄 用目前語言重新執行** — 用同一語言再轉換一次
- **🌐 變更語言並重新執行** — 用另一種語言再轉換一次
- **🗑 清除腳本** — 只刪除自動產生的字幕（你的手動編輯會保留）

![STT 管理選單](images/18-stt-manage-menu.PNG)
*▲ `images/18-stt-manage-menu.PNG` — STT 管理選單（3 個選項）*

> 💡 **提示**：如果辨識準確率太低，會出現「語音辨識準確率可能較低」的提示。這通常發生在音訊的實際語言與你所選語言不同的時候。請用 **Change Language** 按鈕切換語言再試一次。

## 8. 編輯與新增字幕

### 8-1. 手動新增字幕
點擊字幕區域頂部的 **➕ Add Segment** 按鈕，會根據目前播放位置建立一個新片段。在工作表中輸入文字並點擊 **Add** 即可新增卡片。

![Add Segment 工作表](images/19-add-segment.PNG)
*▲ `images/19-add-segment.PNG` — Add Segment 工作表（時間範圍 + 文字輸入 + ▶ 試聽按鈕）*

工作表中的 **▶ 試聽按鈕** 讓你可以短暫試聽那一段。

### 8-2. 編輯字幕
點擊卡片上的 ✏️ 按鈕開啟編輯工作表，可以 **編輯字幕的文字**。用頂部的 ▶ **Loop 按鈕** 可以一邊循環聽該片段一邊聽寫；如果你編輯的是 STT 產生的字幕，下方還會出現 **Original** 和 **Reset** 按鈕，讓你隨時還原成原文。（字幕的時間範圍在這裡無法變更，僅在頂部顯示以供參考。）

![字幕編輯工作表](images/19-edit-segment.PNG)
*▲ `images/19-edit-segment.PNG` — 字幕編輯工作表（編輯文字 + Loop 試聽）*

## 10. Podcast

把你在 Apple Podcasts 訂閱的 Podcast 作為學習素材引入。這是 **進階版功能**。

### 10-1. 開啟
主畫面 ➕ 選單 → **Subscribe to Podcast**。

### 10-2. 從 Apple Podcasts 匯入
首次使用時會看到 **「Load from Apple Podcasts」** 按鈕。點擊它會請求媒體資料庫權限，授權後，你訂閱的 Podcast 會以橫向捲動的卡片顯示。

![Podcast 介面](images/22-podcast.PNG)
*▲ `images/22-podcast.PNG` — Podcast 介面 — Apple Podcasts 訂閱卡片橫向捲動*

點擊頻道卡片，會以工作表開啟它的單集清單。

![單集清單工作表](images/23a-podcast-episodes.PNG)
*▲ `images/23a-podcast-episodes.PNG` — 單集清單工作表（沒有官方字幕的一般單集）*

每個單集會顯示：
- 標題、發布日期、時長
- **Script** 徽章（紫色）— 提供官方時間同步字幕的單集
- **Text only** 徽章（灰色）— 僅有文字的單集
- 無徽章 — 不提供字幕的單集
- 右側狀態：
  - ⬇ 下載圖示 — 尚未下載
  - 🟣 進度環 — 下載中
  - ✅ 綠色勾選 — 下載完成

點擊已勾選的單集即可立即開啟播放器。

![帶 Script 徽章的單集](images/23b-podcast-episodes-script.PNG)
*▲ `images/23b-podcast-episodes-script.PNG` — 帶紫色 **Script** 徽章的單集清單*

> 💡 **為什麼推薦 Script 單集**：帶 **Script** 徽章的單集不使用 iOS 語音辨識（STT）來產生字幕，而是 **直接引入 Podcast 提供的官方字幕**。因為是人工撰寫的準確字幕，比 STT 的錯字和誤辨識少得多，也無需等待轉換。挑選學習素材時，我們建議優先選擇 Script 單集。

### 10-3. 透過 RSS URL 直接匯入
展開畫面底部附近的 **「Add via RSS URL (advanced)」**，即可直接輸入 RSS 位址。用它來匯入 Apple Podcasts 中沒有的 Podcast。

### 10-4. 關於行動數據
預設設定為 **僅在 Wi-Fi 下下載**。如果你在設定裡允許了行動數據下載，那麼在下載較大的單集前會出現確認提示。

## 12. 免費版與進階版

### 永久免費
- 波形顯示 + A-B 循環
- 播放速度調節
- 匯入本機檔案（檔案應用程式）
- 從音樂資料庫匯入
- 書籤（每個檔案最多 3 個）

### 進階版
- 語音轉文字（STT）
- 即時字幕 ↔ 波形同步
- 單字本
- 無限書籤
- Podcast 下載

### 付費方式
**3 天免費試用 → 一次性購買。** 付一次即可終身使用 — 它不是訂閱。即使更換裝置，用同一 Apple ID **「Restore Purchases」** 即可延續。

## 13. 常見問題

**Q. 為什麼我在 Apple Music 上聽的曲目無法匯入？**
A. 透過 Apple Music 訂閱下載的曲目受版權保護（DRM）鎖定，無法在其他應用程式中使用。只有從 iTunes Store 購買或從 CD 匯入的曲目才能使用。

**Q. 語音辨識太不準了。**
A. 檢查兩點：① 音訊的實際語言是否與你所選語言一致，② 是否有大量背景音樂或雜音。對於實在難以辨識的片段，直接用 ➕ Add Segment 或 Enter Manually 自己輸入即可。

**Q. 轉換過程中點了另一個檔案會怎樣？**
A. 會出現 *「切換到 ...？轉換將被中斷。已完成部分會被儲存。」* 的確認。已經完成的部分會被保留。

**Q. 我的資料會透過網際網路傳送嗎？**
A. 不會。Phrabbit 不使用任何外部伺服器。語音辨識使用 Apple 的裝置端功能，你的單字本、書籤和字幕都儲存在你的裝置上。

**Q. 更換裝置後我的資料會怎樣？**
A. 目前版本（v1.0）只把資料儲存在裝置上。在新裝置上安裝應用程式並用同一 Apple ID 執行 **Restore Purchases** 可以解鎖進階版，但單字本、書籤和字幕不會轉移。
